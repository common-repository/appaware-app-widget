The Playboard Android App Widget (formerly AppAware App Widget) allows to easily add Android App details to blog-posts through a minimalistic and easy-to-use HTML widget.

== Description ==

The Playboard Android App Widget allows to easily add Android App details to blog-posts through a minimalistic and easy-to-use HTML widget. To see the widgets in action and more examples go to: http://playboard.me/wordpress-app-widget

The widget is perfect for roundups - see how AndroidPolice is using it (http://www.androidpolice.com/2013/10/24/bonus-round-random-heroes-arena-alpha-hello-hero-and-strategy-tactics-ussr-vs-usa/ ) 

To add the widget to your post use the following short-code templates:

`[pb-app-box pname='AN_ANDROID_APP_PACKAGE_NAME' name='APP_NAME' theme='discover' lang='en']`


The short-code will be replaced with the Playboard App Widget. You can add to the same blog-post as many widgets as you like.

The widget supports multiple languages and themes. If your language is not supported or you need a special theme, feature or even a new plugin contact us at feedback@playboard.me

== Installation ==

This section describes how to install the plugin and get it working.

1. Login to your Wordpress blog
2. Go to Plugins->Add New
3. Search for Playboard
4. Locate the Playboard Android App Widget and click on 'Install Now'
5. Activate the plugin after installation

or manually

1. Download the source for the plugin
2. Upload `playboard_app_widget.php` to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.

You can now use the Playboard App Widget short-code when writing a new blog-post.
